# Learning Objectives

Students can
- Evaluate the style of code.
- Apply a style guide to their own code.
- Use style tools to improve their code:
    - Use a CLI linter
    - Use an IDE linter
    - Use a CLI formatter
    - Use an IDE/Jupyter formatter
- Write code that speaks for itself.
- Import external functions into Jupyter.
