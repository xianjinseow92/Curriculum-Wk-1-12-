# Learning Objectives

Students can:

- Explain the important concepts of Naive Bayes
  - Independence Assumption
  - Complexity
  - Common applications
- Write (pseudocode or whiteboard) the code to execute the NB algorithm.
- Recognize cases where NB would and would not be a good model.
